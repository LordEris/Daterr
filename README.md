# 💌 Daterr

**Un petit site pour demander un rendez-vous — parce que texter c'est trop simple.**

---

### ✨ Comment ça marche ?

| Étape | Ce qui se passe |
|-------|-----------------|
| 💌 | *"Tu veux sortir avec moi ?"* — bouton **Oui** fixe, bouton **Non** qui s'enfuit |
| 🎉 | Elle dit oui → confettis & réaction de panique contrôlée |
| 📅 | Elle choisit une **date** et une **heure** |
| 🍽️ | Elle sélectionne ce qu'on mange |
| 💗 | Résumé du rendez-vous + bouton pour copier & envoyer |

---

### 🚀 Utilisation

Ouvre simplement `index.html` dans un navigateur — aucune dépendance, aucun serveur requis.

```bash
git clone https://github.com/LordEris/Daterr.git
cd Daterr
# ouvre index.html dans ton navigateur
```

---

### 🎨 Détails

- **Bouton "Non"** — fuit la souris, impossible (ou presque) à cliquer. Si tu y arrives, il a une réaction.
- **Progression** — 4 cœurs 💗 qui se colorent étape par étape
- **Résumé final** — date, heure et repas choisis, copiables en un clic
- **Tout en français** — évidemment
